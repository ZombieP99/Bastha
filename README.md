## 🇬🇧 English

Welcome to **Bastha** (بسطة)! Bastha is a premium, beautifully designed e-commerce and local shop discovery platform. It aims to connect users with the best local stores, offering a seamless browsing experience with full bilingual support.

### ✨ Key Features

| Feature | Description |
|---------|-------------|
| 🌍 **Bilingual Interface** | Full i18n support offering content in both English and Arabic. |
| 🔄 **RTL Architecture** | Automatic layout flipping and tailored typography for Right-to-Left languages. |
| 🎨 **Premium UI/UX** | Built with Tailwind CSS and Framer Motion for buttery-smooth animations. |
| 🔎 **Dynamic Discovery** | Real-time search and category filtering for shops and stores. |
| 🚀 **High Performance** | Powered by Next.js App Router and Turbopack for instantaneous load times. |

### 🛠️ Detailed Setup & Installation Guide

Follow these precise steps to download, install, and run the project locally on your machine.

#### Step 1: Prerequisites
Before you begin, ensure you have the following tools installed:
1. **[Node.js](https://nodejs.org/en/)**: (Version 18 or higher). Download and install it from the official website.
2. **Code Editor**: We highly recommend [VS Code](https://code.visualstudio.com/).
3. **Git**: (Optional) For cloning the repository via the command line.

#### Step 2: Download the Project
You can download the project files in one of two ways:

**Method 1: Using Git**
Open your Terminal or Command Prompt and run the following command:
```bash
git clone https://github.com/ZombieP99/Bastha
```

**Method 2: As a ZIP File**
- Click the green **"Code"** button at the top of this GitHub page.
- Select **"Download ZIP"**.
- Once downloaded, extract (unzip) the folder anywhere on your computer.

#### Step 3: Install Dependencies
Open the extracted project folder in your terminal (or inside VS Code's integrated terminal), and run the following commands:

```bash
# 1. Navigate into the project directory
cd projact

# 2. Install all required packages and dependencies
npm install
```
*(Note: This process might take a few minutes depending on your internet connection).*

#### Step 4: Run the Application
Everything is set up! To start the local development server, run:

```bash
# Start the development server
npm run dev
```

Once you see the success message in your terminal, open your favorite web browser and visit:
👉 **[http://localhost:3000](http://localhost:3000)**

### 🛠️ طريقة التنزيل والتشغيل بالتفصيل

اتبع هذه الخطوات الدقيقة لتنزيل المشروع وتشغيله على جهازك الشخصي بنجاح.

#### الخطوة 1: المتطلبات الأساسية (Prerequisites)
قبل البدء، تأكد من تثبيت البرامج التالية على جهازك:
1. **[Node.js](https://nodejs.org/en/)**: (الإصدار 18 فما فوق). قم بتحميله وتثبيته من الموقع الرسمي.
2. **محرر أكواد**: ينصح باستخدام [VS Code](https://code.visualstudio.com/).
3. **Git**: لنسخ المشروع باستخدام سطر الأوامر.

#### الخطوة 2: تنزيل المشروع (Download)
يمكنك الحصول على ملفات المشروع بإحدى الطريقتين:

**الطريقة الأولى: باستخدام Git (مستحسن)**
افتح سطر الأوامر (Terminal) واكتب الأمر التالي:
```bash
git clone https://github.com/ZombieP99/Bastha
```

**الطريقة الثانية: كملف مضغوط (ZIP)**
- اضغط على الزر الأخضر **"Code"** في أعلى صفحة المشروع على GitHub.
- اختر **"Download ZIP"**.
- بعد التحميل، قم بفك الضغط (Extract) عن الملف في أي مكان بجهازك.

#### الخطوة 3: تثبيت الحزم (Installation)
افتح مجلد المشروع في سطر الأوامر (أو داخل نافذة Terminal في VS Code)، وقم بتنفيذ الأوامر التالية:

```bash
# 1. الدخول إلى مجلد المشروع
cd projact

# 2. تثبيت الحزم والاعتمادات البرمجية اللازمة
npm install
```
*(ملاحظة: قد تستغرق هذه العملية بضع دقائق حسب سرعة الإنترنت لديك).*

#### الخطوة 4: تشغيل الموقع (Running)
الآن كل شيء جاهز! لتشغيل الموقع، اكتب الأمر التالي:

```bash
# تشغيل خادم التطوير المحلي
npm run dev
```

بمجرد ظهور رسالة النجاح في الشاشة، افتح متصفح الإنترنت الخاص بك وانتقل إلى الرابط التالي:
👉 **[http://localhost:3000](http://localhost:3000)**
